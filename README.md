## 📌 О проекте

Проект представляет собой аналитический дашборд для сервиса доставки еды **«Всё.из.кафе»** в городе **Саранск** за период **1 мая – 30 июня 2021 года**.

**Цель** – оценить эффективность работы сервиса через ключевые продуктовые и бизнес-метрики: активность пользователей (DAU), конверсию в заказ (CR), средний чек, удержание (Retention Rate) и LTV ресторанов-партнёров.

Дашборд включает графики, таблицы и аналитическую записку с выводами и рекомендациями.

## 📈 Ключевые метрики и визуализации

| Метрика | Описание | Тип визуализации |
|---------|----------|------------------|
| **DAU** | Ежедневное количество уникальных пользователей | Линейная диаграмма |
| **Conversion Rate (CR)** | Доля пользователей, сделавших заказ | Линейная диаграмма |
| **Средний чек** | Средняя выручка с заказа (комиссия / заказы) | Таблица |
| **Retention Rate** | Доля пользователей, вернувшихся в 1–7 дни | Линейная диаграмма + таблица |
| **LTV ресторанов** | Суммарная комиссия с заказов по сети | Таблица (топ-3) |
| **Популярные блюда (по LTV)** | Блюда в топ-2 ресторанах с наибольшим LTV | Таблица (топ-5) |

## 🧠 Основные выводы

### ✅ DAU
- Пик – **85 пользователей** (в начале мая)
- К концу июня активность снижается до **30–40 пользователей**
- В праздники (9 мая) – всплеск активности

### ✅ Conversion Rate
- Максимум – **43%** (4 мая)
- Минимум – **18%** (24 июня)
- Тенденция к росту в конце июня (до 37%)

## 📎 Рекомендации

### По DAU и удержанию
- Запустить **программу лояльности** для пользователей, которые не заказывали 5–7 дней
- Провести **A/B-тест push-уведомлений** в рабочие дни
- Проанализировать воронку для выявления причин низкого удержания (доставка, ассортимент, цена)

### По Conversion Rate
- **Проверить технические ошибки** 24 июня (минимальная конверсия 18%)
- **Запланировать акции на майские праздники** в следующем году

### По LTV ресторанов
- Предложить «Шоколадному Раю» **совместную акцию** или пересмотреть комиссию
- **Усилить продвижение** топ-2 ресторанов (выделить в поиске, добавить в рекомендации)

### По популярным блюдам
- **Расширить мясную линейку** в обоих топ-ресторанах
- Проверить, почему острые блюда не в топе (региональная особенность)
